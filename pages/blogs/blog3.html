    • HTML & CSS

    Among the key technologies for building Web pages are HTML (the Hypertext Markup Language) and CSS (the Cascading Style Sheets). HTML provides the structure of the page, CSS the layout (visual and audio

    The HTML language is used to describe the structure of Web pages. HTML allows authors to: Create online documents with headings, text, tables, lists, photos, etc.

    A CSS file describes how color, layout, and fonts are presented on a Web page. CSS can be used independently of HTML and can be used with any XML-based markup language. It enables the presentation to adapt to different types of devices, such as large screens, small screens, or printers.

    As a result of separation of HTML from CSS, websites are easier to maintain, style sheets can be shared across pages, and pages can be tailored to different environments. This is known as structure (or: content) from presentation.

    Web browser receive HTML document from web server and render document into web pages. HTML help to describe web pages semantically. On the other hard CSS is like the adjective in a sentence. It helps to to decorate web pages as your wish.

    HTML elements introduce content to the page using tags in between angle brackets. It has it's own syntax to write code and run it properly.
